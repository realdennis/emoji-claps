# Emoji-Claps

A custom element which is experience-addicted for clap / like / dislike.

## Usage

```html 
<emoji-claps 
             emoji="👏" 
             currentcount="30" 
             maxcount="50" 
             bullets='["😍","🎉","🔥","😆"]' 
             bulletcount="6" 
             prefix="+" />
```
